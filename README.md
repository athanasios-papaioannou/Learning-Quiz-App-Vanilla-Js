# 📝 Learning Quiz

A Vanilla JavaScript quiz application inspired by Learning Management Systems (LMS). Built as a case study to strengthen core frontend development concepts such as state management, asynchronous data loading, and dynamic DOM rendering.

---

## ✨ Features

- Dynamic quiz rendering
- External JSON question database
- Progress tracking
- Score calculation
- Success percentage
- Previous / Next navigation
- Answer validation
- Final results screen
- Responsive design

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- JSON
- Fetch API

---

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/athanasios-papaioannou/Learning-Quiz-App-Vanilla-Js.git
```

Open the project

```bash
cd Learning-Quiz-App-Vanilla-Js
```

Launch

Use **Live Server** (recommended) or any local development server to run the application.

## 🧠 Key Learnings

During this project I focused on understanding how small JavaScript applications are structured, moving beyond isolated functions and thinking in terms of application flow and user interaction.

Some of the concepts explored include:

- Application State Management
- DOM Manipulation
- Event Handling
- Dynamic DOM Rendering
- Fetch API Integration
- JSON Data Handling
- User Input Validation
- Score & Progress Tracking
- Responsive UI Design
